# a7
a7 is a small note taking app designed for modern journaling in a digital world.

## Screen map

Flow:
Welcome → Walkthrough Storage → Walkthrough Privacy → Setup Review → Dashboard

From Dashboard:
- enter → Viewer
- n → Editor (new)
- e → Editor (edit selected)

Viewer:
- e → Editor (edit current)
- esc → Dashboard

Editor:
- ctrl+s save
- esc → Dashboard

Screens:
1) Welcome
2) Walkthrough Storage
3) Walkthrough Privacy
4) Setup Review
5) Dashboard
6) Viewer
7) Editor
8) Settings/About (planned)
